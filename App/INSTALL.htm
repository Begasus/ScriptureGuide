Scripture Guide Installation

This is for the installation from the sources. The sources are released under GNU Public License.

This version is meant for developers. If you only want to use Scripture Guide, then get the binary package from the Scripture Guide Website.

I. Compiling and installing Scripture Guide

This software is only for Haiku. It will not compile on any other platform. It also requires the SWORD library (which can be built using haikuporter.
  • Run make in the ScriptureGuide and ScriptureGuideManager directories and copy the App directory to wherever you want.
  • Use ScriptureGuideManager to download books.
