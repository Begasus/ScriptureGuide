Scripture Guide Installation

This is for the installation from the sources. The sources are released under GNU Public License.

This version is meant for developers. If you only want to use Scripture Guide, then get the binary package from the Scripture Guide Website.

I. Compiling and installing Scripture Guide

This program is for BeOS, Zeta, and Haiku. It will not compile on any other platform. It also requires the Sword library.
  1. Building the SWORD Library (written for 1.5.7)
    1. Download the source archive from the SWORD Project's web page at www.crosswire.org
    2. Unzip the archive to /boot/home
    3. Edit src/utilfuns/zlib/untgz.c: change line 14 from #ifdef unix to #ifndef unix
    4. Edit usrinst.sh line 8 from OPTIONS="--enable-debug $OPTIONS" to #OPTIONS="--enable-debug $OPTIONS"
    5. Create a symlink in /boot/develop/lib called libstdc++.so which points to libstdc++.r4.so
    6. run userinst.sh from the Terminal
    7. run make; make install
  2. Go to the Scripture Guide directory, run make, and move the directory, complete with sources, to wherever you want on the hard drive, and run make.
  3. Download the modules you like from http://www.crosswire.org/sword/modules
  4. Create the folder /boot/home/config/add-ons/scripture-guide and extract all modules to this folder
