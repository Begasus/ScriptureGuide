ScriptureGuide 0.9.0

Welcome to ScriptureGuide, the bible study program for Haiku.

Current features include:

  • Show bibles and commentaries
  • Search a module or a subset of a module
  • Additional fancy stuff like help, take notes, set font etc.

Planned features:

  • Write a custom GBF parser.
  • Search in more than one module at once.
  • Support for Strong numbers.
  • Support for all types of modules (not only Bibles and commentaries).

Version 0.9.0 known bugs are:

  • Hebrew is displayed from left to right, this due to the limitations of BTextView
  • Riegers comment is not converted to UTF8 (problem in SWORD?)
  • Saving preferences is buggy.
