Scripture Guide 0.8.0

Welcome to Scripture Guide, the bible study program for BeOS, Zeta and Haiku. (Note: in version 0.8.0 the program's name was changed from Be-Logos to Scripture Guide due to copyright reasons.)

This is the source package for developers. Feel free to modify the code and to contribute features and bugs alike. Contact the Scripture Guide team at The Scripture Guide website.

See the following files for additional information:
LICENSE - the GNU Public License
INSTALL - installation instructions for Scripture Guide

Current features include:

  • Show bibles and commentaries
  • Search a module or a subset of a module
  • Additional fancy stuff like help, take notes, set font etc.

Planned features:

  • Write a own GBF parser
  • Search in more than one module at once
  • Support for Strong numbers
  • Support for all types of modules (not only bibles and comments)

Version 0.8.0 known bugs are:

  • Hebrew is displayed from left to right, this due to the limitations of BTextView
  • Riegers comment is not converted to UTF8 (problem in Sword?)
  • saving preferences has errors
